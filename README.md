## Featured Work

### [`codex-local-sdk-python`](https://github.com/maestromaximo/codex-local-sdk-python)

Python SDK-style wrapper for Codex non-interactive local execution.

Includes sync/async execution, JSONL event parsing, resumable sessions, retry/backoff controls, observability hooks, schema-constrained output support, tests, CI, and PyPI packaging.

**Stack:** Python · Codex CLI · PyPI · GitHub Actions · Testing · Developer Tooling

---

### [`agents-sdk-mutta`](https://www.agarciap.com/book/)

CLI scaffolding tool for OpenAI Agents SDK projects.

Mutta provides an opinionated project structure for multi-agent services, including manager-based orchestration, one-agent-per-file conventions, Pydantic-first outputs, and AI-readable project rules for coding assistants.

**Stack:** Python · OpenAI Agents SDK · CLI Tooling · Agent Architecture · Pydantic

---

### [`FastChapter`](https://github.com/maestromaximo/FastChapter)

Voice-first desktop studio for drafting books chapter by chapter.

FastChapter is a local-first Electron app for writers that combines voice recording, transcription, structured book workspaces, LaTeX scaffolding, Codex-assisted writing workflows, PDF preview, and desktop packaging.

**Stack:** TypeScript · React · Electron · Vite · OpenAI Transcription · LaTeX · Desktop Apps

---

### [`forgelore`](https://github.com/maestromaximo/forgelore)

Autonomous research copilot project exploring how agentic systems can help plan, gather, synthesize, and structure complex research.

**Stack:** Python · Research Automation · Knowledge Synthesis · Agentic AI
